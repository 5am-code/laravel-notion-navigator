The package [5am-code/laravel-notion-api](https://github.com/5am-code/laravel-notion-api/edit/main/README.md) allows you, to easily access the new Notion API, fetch pages/databases, read properties etc.

This small project showcases some of the basic features. 
Feel free to play around and discover the power of the Notion API.
